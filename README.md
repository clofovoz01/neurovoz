# neurovoz
NeuroVoz IA: Ecosistema de Rehabilitación Vocal Neurocognitiva  SaaS médico basado en IA para el análisis de biomarcadores de voz (Jitter/Shimmer). Integra una Web App con pagos vía PayPal, base de datos en Supabase y App nativa en React Native.Diseñado para automatizar el seguimiento clínico y la reprogramación fonatoria mediante biofeedback real.
